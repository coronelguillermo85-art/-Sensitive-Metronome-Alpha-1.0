### 📋 Descripción del Proyecto
El **Sensitive Metronome** es una herramienta de luthería digital de código abierto diseñada para músicos que requieren una referencia de pulso precisa y visible. Este prototipo, desarrollado bajo el protocolo de investigación de **Sensitive Lab**, responde a la necesidad de un seguimiento rítmico visual y sonoro en ensayo y ejecución en vivo.

A diferencia de los metrónomos estándar, este dispositivo utiliza una pantalla **OLED SSD1306** para mostrar el tempo en **BPM** y la estructura del **compás en tiempo real**.

### 🚀 Funciones Principales

- **Control de BPM Dinámico:** ajuste mediante potenciómetro en rango de 40 a 200 BPM.
- **Monitor de Compás Visual:** visualización del tiempo activo en gran tamaño.
- **Identidad Sensitive Custom:** estética de laboratorio integrada en la interfaz.
- **Feedback Dual Sincronizado:** click sonoro y referencia visual del pulso.
